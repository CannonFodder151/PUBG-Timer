Please follow this setup guide for PUBG Timer V1.0

1. Extract all files in to your chosen location
2. Run the "setup.exe" file
3. Make a shortcut of the "PUBG Timer Program" file and place it where you choose

Make sure to check www.PUBGTimer.weebly.com for updates and to report bug fixes for the program
